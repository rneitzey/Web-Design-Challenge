Summary: Latitude vs. X

The purpose of this project was to analyze changes in meteorological phenomena in relation to proximity to the equator. To accomplish this analysis, we pulled data from the OpenWeatherMap API in order to compile a dataset for 500 cities.

After assembling the data set, we used Matplotlib to plot various aspects of the weather versus latitude. Factors we looked at included temperature, cloudiness, winds speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations.
